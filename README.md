#React + Webpack
##Install
_For more detail: [package.json dep + devDep](./package.json)_
```
npm i
```
##Run
_For more detail: [package.json scripts](./package.json)_
```
npm run dev
or
npm run prod
```

##Build
###Adding types
[typings.json](./typings.json)
```
Globally install typings
npm i -g typings
Example
typings install --global --save dt~react
```
